Les routes : 

EPI 
- Voir tous les epis :
GET api/epis
- Modifier les epis :
PUT api/epis/:id
- supprimer les epis :
DELETE api/epis/:id
- Ajouter un contrôle :
POST api/epis/:id/controle

CONTROLES
- Voir les controles :
GET api/controles/:id
- Supprimer les controles
DELETE api/controles/:id
- Ajouter un contrôle :
PUT api/controles/:id

ALERTES :
GET api/alert

LOGIN :
POST /api/login
